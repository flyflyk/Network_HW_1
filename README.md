## 使用方法

### 安裝依賴
```bash
pip install -r requirements.txt
```

### Data Link Layer 實驗
以管理員身分執行:
```bash
python raw.py "<介面卡名稱>" <目標 MAC 地址>
```

### Network Layer 實驗
以管理員身分執行:
```bash
python ip.py <目標 IP 地址>
```
